🕵️ Detective Story Game — FINAL COMPLETE (JavaFX + Maven)

✅ RUN (IntelliJ)
1) File -> Open -> select this project folder
2) Maven tool window -> Plugins -> javafx -> javafx:run

✅ RUN (Terminal)
mvn clean javafx:run

HOW TO PLAY
- Actions start at 6.
- Locations tab: select a location -> SEARCH LOCATION (costs 1 action) -> collect evidence.
- Suspects tab: Interview (costs 1 action) to see statements.
- Present Evidence (FREE): select suspect + evidence to expose contradictions.
- Accuse tab: choose suspect and accuse when ready.

If you see “mvn is not recognized”, install Maven or use IntelliJ Maven run.
